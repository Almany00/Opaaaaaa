ما هي HTML؟

HTML هي لغة ترميز تستخدم لإنشاء وتصميم صفحات الويب. تعني "HyperText Markup Language"، وتستخدم لوصف هيكل المحتوى على الويب. تتكون HTML من مجموعة من العناصر (Elements) التي تحدد كيفية عرض المحتوى في المتصفح.

مكونات HTML الأساسية:

  • الوسوم (Tags): مثل <h1> للعناوين و <p> للفقرات.
  • الخصائص (Attributes): مثل src للصور و href للروابط.
  • الهيكل الأساسي: كل صفحة HTML تبدأ بـ <!DOCTYPE html> وتحتوي على <html>, <head>, و <body>.
            
<!DOCTYPE html>
<html>
<head>
    <title>عنوان الصفحة</title>
</head>
<body>
    <h1>مرحبًا بالعالم!</h1>
    <p>هذا مثال بسيط لصفحة HTML.</p>
</body>
</html>

ما هي CSS؟

CSS هي لغة تنسيق تستخدم لتزيين وتصميم صفحات الويب. تعني "Cascading Style Sheets"، وتتحكم في الألوان والخطوط والمسافات والتخطيط العام للصفحة. يمكن استخدام CSS داخليًا (ضمن ملف HTML) أو خارجيًا (في ملف منفصل).

مكونات CSS الأساسية:

  • القواعد (Rules): تتكون من "محدد" (Selector) و "إعلان" (Declaration).
  • الخصائص (Properties): مثل color للألوان و font-size لحجم الخط.
  • القيم (Values): مثل red للألوان أو 16px للأحجام.
            
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    color: #333;
}

h1 {
    color: #0066cc;
}
